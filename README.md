# label-generator

Gerador de etiquetas de preço em PDF para folhas Pimaco A5Q-813 (13 × 14 = 182 etiquetas por folha).

Um projeto pessoal para facilitar a impressão de etiquetas de preço sob demanda, direto pelo celular, sem depender de softwares específicos.

## Como rodar

```bash
pip install -r requirements.txt
python app.py
```

Acesse em `http://localhost:4000`

## Docker

```bash
docker compose up -d
```
